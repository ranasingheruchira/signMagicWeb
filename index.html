Literature Survey

A comprehensive literature review is a necessary stage in every research study, and the "Sign Magic" program is no exception. In recent years, the field of machine learning-based sign language detection has expanded dramatically. However, selecting an appropriate model for efficient sign language detection remains a significant challenge. Many researchers contributed to overcome these issues by using various techniques. Some of the most relevant approaches used by the previous researchers are described below.

  • "Sign Language Recognition Based on Computer Vision" IEEE International Conference on Artificial Intelligence and Computer Applications 2021

    A conferrence proceeding published by W.Li, H.Pu, and R.Wang on IEEE International Conference on Artificial Intelligence and Computer Applications in 2021 describes use of Convolutional Neural Networks and LSTM in Sign Language recognition and their potential costs and benifits.

  • "British Sign Language Recognition using HOG" International Students' Conference on Electrical, Electronics and Computer Science, 2018

    A conferrence proceeding published by R.D.Raj and A.Jasuja on International Students' Conference on Electrical, Electronics and Computer Science describes use of Artificial Neural Networks in BSL (British Sign Language) recognition.

  • "American sign language recognition in game development for deaf children" 8th International ACM SIGACCESS Conference on Computers and Accessibility

    This research conducted based on the Hidden Markov Model for sign language recognition. By using this technique, H.Brashear, Z.Zafrulla, P.Yin, P.Presti, T.Starner and H.Harley Hamilton created a "Copy Cat" which responds to the American sign language.

Research Overview and Research Gap

This research focuses on utilizing computer vision in educational games to improve the learning experience of hearing-impaired primary school students. The main objective of this study is to compare and select the best video classification models that can be used to recognize sign language related to the primary school curriculum and employ them in the context of mobile games.

According to the above literature, multiple types of research have been conducted to recognize sign language using Deep Learning techniques. However, only a limited number of researchers are focused on utilizing them in visual recognition games. Also, the above researches are mainly focused on developing sign language recognition algorithms to classify sign language related to the English alphabet. So, there is a limitation in recognizing numeric characters and mathematical terms related to sign language variants. Also, the performance of the above models may not be heavily taken into consideration. Since mobile games require high performance, it is a must to select the optimal sign language recognition algorithm based on performance and resource consumption. The most important novel concept of this research component is the utilization of Deep Learning in visual recognition games to teach mathematics to hearing impaired primary school students, which has not completed in this domain before.

Research Problem

This research focusses on addressing the following research problems,

  • "How visual recognition games can be used to improve the learning expirience of hearing impaired primary school students?
  • "What are the most efficient video classification model to evaluate language signs?"

Research Objectives

The main objective of this research is to develop a visual recognition game to assist and evaluate the learning process of hearing-impaired primary school students by using a series of image classification models that can recognize sign language related to the Sri Lankan primary school curriculum. Considering the complexity of the main objective, it is future divided into specific objectives as follows.

  • Develop mathematical game component to evaluate mathematical skills of hearing impaired students such as Number Recognition, Counting, Basic mathematical Operations, and Complex Mathematical Operations.

  • Develop sinhala sign language game component to evaluate Sinhala language skills of hearing impaired students related to Sinhala Letter Recognition and Grammer.

  • Develop environment skills game component to evaluate environment skills of hearing impaired students such as Plant Recognition, Astronomical object recognition, and, recognition of animals in surrounding.

  • Develop General Skills game component to teach sign language signs related to general skills such as greetings, and asking for directions etc.

Methodology and System Overview


The methodology for developing the Sign Magic Mobile Game involves,

  • Collecting a diverse ataset of sign videos

  • Creating and evaluating different video classification models

  • Creating a user-friendly interface with React Native

  • Ensuring scalability and real-time analysis using cloud-based infrastructure

Technologies

Flask

Flask is a lightweight and versatile Python web framework known for its simplicity and ease of use. It is widely used for building web applications, APIs, and microservices. Flask's minimalist design and extensive ecosystem of extensions make it a popular choice for web developers seeking flexibility and rapid development.

Tensorflow

TensorFlow is a free and open-source software library for dataflow and differentiable programming across a range of tasks. It is a symbolic math library, and is also used for machine learning applications such as neural networks.

React Native

React Native is an open-source UI software framework created by Meta Platforms, Inc. It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web, Windows and UWP by enabling developers to use the React framework along with native platform capabilities.

Keras

Keras is an open-source library that provides a Python interface for artificial neural networks. Keras acts as an interface for the TensorFlow library. Up until version 2.3 Keras supported multiple backends, including TensorFlow, Microsoft Cognitive Toolkit, R, Theano, and PlaidML.
